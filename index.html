What's the weather like?

The intent of this project was to create a Python script to visualize the weather of 500+ cities across the world of varying distances from the equator. In order to complete the analysis we made API calls to OpenWeatherMap and captured data points of over 500 cities.

We built a series of scatter plotts using the MatPlot library to showcase the following relationships:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
